# Product Hunt packet: Galley

## Name
Galley

## Tagline (60 max)
A beautiful, free Markdown reader. Set it as your default.

## Description (260 max)
I made a beautiful, free Markdown reader for the Mac. Double-click a .md
file and get a clean, typeset page instead of raw text. Read-only by
default, with a quick edit mode when you need it. Five themes, Mermaid,
math. Open source, no tracking. Set it as your default.

## First comment (maker story, post immediately at launch)

Hey, I made a free Markdown reader for the Mac.

Double-click a .md file and it just renders nicely instead of showing
raw text or opening in your code editor. It's read-only by default
(there's a quick edit mode if you need to fix something), has five
themes, and renders Mermaid diagrams and math.

I built it because I wanted something simple to set as my default .md
viewer and nothing quite fit. It's free, open source, and doesn't track
anything.

Would love for you to try it and let me know what you think.
